# Projeto_ETL_SantanderDevWeek-
Explorando IA Generativa em um Pipeline de ETL com Python


*Aluna: Idna da Silva Cunha dos Reis*


Projeto de ETL (Extração, Transformação e Carregamento) desenvolvido para demonstrar conceitos de Ciência de Dados com Python.


## FLUXO DO PROJETO:

### EXTRAÇÃO
Coleta dados de usuários e suas contas bancárias.

### TRANSFORMAÇÃO
Define status da conta (ATIVA ou BLOQUEADA PARA OPERAÇÕES).

### CARREGAMENTO
Salva dados tratados no arquivo usuarios_etl_final.txt.
